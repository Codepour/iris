# Functionalities and Statistical Tests

## Data Management
- **Import**: CSV file loading with automatic type detection (Nominal/Ratio).
- **Editing**: Spreadsheet interface for viewing and editing cell values.
- **Selection**: Support for row, column, cell, and range selections.
- **Missing Values**: Options for Listwise and Pairwise deletion.

## Statistical Analysis

### Descriptive Statistics
- **Central Tendency**: Mean, Median, Mode.
- **Dispersion**: Variance, Standard Deviation, Range, Min, Max, IQR.
- **Distribution**: Skewness, Kurtosis.
- **Percentiles**: Quartiles (Q1, Q3) and customizable percentiles.

### Distribution Analysis
- **Frequency Table**: Bin-based frequency counts.
- **Visualization**: Histogram charts.
- **Outliers**: Detection based on Z-score (> 3).

### Correlation
- **Methods**: Pearson, Spearman.
- **Significance**: One-tailed and Two-tailed tests.
- **Partial Correlation**: Pearson method with control variables.

### Distances
- **Metrics**: Euclidean, Squared Euclidean, Manhattan, Chebyshev.
- **Output**: Distance matrix between cases.

### Linear Regression
- **Model**: Ordinary Least Squares (OLS).
- **Fit Statistics**: R, R², Adjusted R², Standard Error of Estimate.
- **Coefficients**: Unstandardized (B), Standardized (Beta), Standard Errors.
- **Inference**: t-statistics, p-values, 95% Confidence Intervals.
- **Diagnostics**: Residuals and Predicted Values.
